About Master Crops

Master Crops are templates that define which crops farmers can list on the platform. When you create a master crop:

  • Farmers can select it when posting their produce
  • It helps maintain consistency across listings
  • Buyers can easily search and filter by crop type
Available Categories
  • Vegetables - Tomatoes, Peppers, etc.
  • Fruits - Apples, Oranges, etc.
  • Grains - Rice, Wheat, etc.
  • Pulses - Lentils, Beans, etc.
  • Spices - Turmeric, Chili, etc.
  • Cash Crops - Cotton, Sugarcane, etc.
  • Other - Miscellaneous crops
